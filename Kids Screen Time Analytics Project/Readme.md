# 📊 Kids Screen Time Analytics Dashboard | Power BI Project

An interactive Power BI dashboard designed to analyze children's screen usage patterns, behavioral trends, and associated health risks. This project transforms raw data into meaningful insights to support better decision-making for parents, educators, and policymakers.

---

## 🚀 Project Overview

Excessive screen time among children is a growing concern. This dashboard provides a data-driven analysis of screen usage, helping to identify:

- Overuse patterns  
- Behavioral trends (Education vs Recreation)  
- Device dependency  
- Health risks linked to screen exposure  

---

## 📌 Key Metrics

- 👶 **Total Kids Analyzed:** 10K+  
- ⏱️ **Average Screen Time:** 4.37 hrs/day  
- ⚠️ **% Exceeding Limit:** 86%  
- 🚨 **High Risk Users:** ~6%

### 🔹 5. Insights
- 86% of children exceed recommended screen limits
- Recreation dominates over educational usage
- Smartphones are the primary driver of screen time
- High screen time strongly correlates with health issues
- Teens are the most vulnerable high-risk group
- Usage patterns are consistent across gender and location

---

## 🔍 Key Insights

- 📊 Majority of children exceed safe screen time limits  
- 🎮 Recreational usage is the dominant contributor  
- 📱 Smartphones drive maximum engagement  
- ⚠️ Increased screen time leads to health risks  
- 👥 Gender and location have minimal impact  
- 🎯 High-risk users are concentrated in specific segments  

---

## 💡 Recommendations

- Set age-based screen time limits  
- Encourage educational content over recreational usage  
- Monitor smartphone usage  
- Focus on high-risk groups (Teens)  
- Promote healthy habits (sleep, physical activity)
