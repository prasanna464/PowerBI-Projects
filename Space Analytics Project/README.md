# 🚀 Global Space Mission Analytics Dashboard

## 🌌 Story Behind This Project

When I was a child, I was always curious about **space, rockets, and airplanes** — how rockets launch and how humans explore space. Like many kids, I dreamed of becoming many things, and one of those dreams was becoming an **astronaut or scientist**.

While working on my **Power BI internship project**, I came across a dataset about **global space missions**. I chose it because it connected with that childhood curiosity.

This project allowed me to combine my **interest in space** with **data analytics**, transforming raw mission data into **meaningful insights through visualization**.

---

# 📊 Project Overview

This project is an **interactive Power BI dashboard** analyzing **historical space missions from 1957 to 2020**.

### Key Goals
* Analyze the **growth of space missions over time**
* Understand **mission success and failure rates**
* Evaluate **company performance in space launches**
* Study **rocket usage and reliability**
* Explore **global launch locations and activities**

The main objective was to **transform raw mission data into clear visual insights using Power BI**.

---

# 📂 Dataset

**Source:** Kaggle – *All Space Missions Dataset*

### Dataset Includes
* Mission Date
* Rocket Name
* Company Name
* Launch Location
* Country
* Mission Outcome
* Rocket Status
* Mission Cost

**Total Records:** `4324 Space Missions`

---

# 🧹 Data Cleaning & Transformation

Before building the dashboard, the dataset was cleaned using **Power Query**.

### Key Data Preparation Steps
* Removed unnecessary columns
* Handled missing values
* Converted data types
* Extracted **year from mission date**
* Standardized **mission outcome categories**
* Cleaned **company names and rocket details**

These steps ensured the dataset was **ready for analysis and visualization**.

---

# 📊 Dashboard Pages

The final dashboard contains **three analytical pages**.

---

# 1️⃣ Executive Overview

Provides a **high-level summary of global space mission activity**.

### Key Metrics
* Total Missions
* Total Companies
* Successful Missions
* Total Investment

### Visualizations
* Space Launch Trend Over Time
* Mission Success Rate
* Top 10 Space Launch Companies
* Space Mission Investment Trend

A **year filter** allows users to explore different **time periods**.

---

# 2️⃣ Company & Rocket Performance Analysis

This page focuses on **rocket usage and company performance**.

### Key Visualizations
* Most Frequently Used Rockets
* Rocket Popularity Over Time
* Rocket Reliability *(Success vs Failure)*
* Mission Outcomes by Rocket Operational Status

### Additional Insights
* Most Active Launch Company
* Most Used Rocket

Users can filter results by:

* Company
* Rocket Type

---

# 3️⃣ Global Launch Geography

This page analyzes the **geographical distribution of space launches**.

### Key Insights
* Most Active Launch Country
* Total Launch Sites

### Visualizations
* Global Launch Distribution Map
* Top Countries by Launch Activity
* Most Active Launch Sites
* Launch Activity Over Time
